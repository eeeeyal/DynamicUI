# Scripts Directory

תיקייה זו מכילה את כל קבצי ה-PowerShell (.ps1) של הפרויקט.

## רשימת Scripts:

### התקנה והגדרה:
- **`install_flutter.ps1`** - התקנת Flutter SDK
- **`add_flutter_to_path.ps1`** - הוספת Flutter ל-PATH

### המרה ויצירת קבצים:
- **`convert_to_static.ps1`** - המרת HTML לקבצים סטטיים
- **`create_example_zip.ps1`** - יצירת ZIP דוגמה
- **`create_test_zip.ps1`** - יצירת ZIP לבדיקות
- **`create_test_gz.ps1`** - יצירת GZ לבדיקות
- **`create_caspit_zip.ps1`** - יצירת ZIP עבור CASPIT
- **`analyze_html_gz.ps1`** - ניתוח קבצי HTML/GZ
- **`fix_html_clicks.ps1`** - תיקון קליקים ב-HTML

## שימוש:

הרץ כל script עם:
```powershell
.\scripts\<script-name>.ps1
```
